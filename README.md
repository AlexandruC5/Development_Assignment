# Salty the Slime

### Controls
	-A/D: Horizontal movement.

	-Space bar: Jump

	-Hold Spacebar: Charged jump.
	
	Debug Keys:
	
	-F1: Load first scene.

	-F2: Reload current scene.

	-F3: Load next scene.

	-F5: Save.

	-F6: Load.

	-F9: Colliders Draw.

	-F10: God Mode.


When in God Mode:
	
	-Space bar: Fly up.

	-S: Fly down.

	-A/D: Horizontal movement.
