# Development_Assignment

### Controls
-A/D: Horizontal movement.
-Spacebar: Jump
-Hold Spacebar: Charged jump.

When in God Mode:
-Spacebar: Fly up.
-S: Fly down.
-A/D: Horizontal movement.

### Debug Keys
-F1: Load first scene.
-F2: Reload current scene.
-F3: Load next scene.
-F5: Save.
-F6: Load.
-F9: Colliders Draw.
-F10: God Mode.
